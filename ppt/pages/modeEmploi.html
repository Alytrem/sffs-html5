Bienvenue sur la démonstration HTML Capgemini

Introduction

Vous êtes dans l'environnement virtuel de démonstration de la technologie HTML5. Utilisez les flèches directionnelles de votre clavier pour vous déplacer, et votre souris pour changer la direction de votre regard (clic + déplacer).

Les salles

Plusieurs salles ont étés mises en place dans l'environnement 3D. Chacune a une fonction bien particulière, comme démontrer une fonctionnalité offerte par HTML5.
Voici la liste des salles et ce qu'elles permettent.

  • Salle LocalStorage : permet d'essayer la fonction de stockage local (clé -> valeur) de HTML5.

La technique

Cette démonstration s'adapte aux capacités de votre navigateur. Bien entendu, l'idéal est un navigateur réçent compatible HTML5 comme Google Chrome. Cependant, une détection des capacités du navigateur est utilisée afin d'afficher l'environnement avec webGL, ou flash si le navigateur ne supporte pas la balise canvas et le WebGL.
Concrètement, l'environnement 3D est toujours affiché (sous réserve qu'au moins flash soit installé)

  • Technologies : Flash & HTML5 (incluant CSS3, javascript et HTML5)
  • Langages : javascript, HTML5, CSS (sans inclure les langages du code généré (ActionScript...)
  • Framework : Copperlicht (pour piloter webgl)
  • Logiciels : Sketchup (pour créer les meshs 3D), Coppercube (pour créer la scène, caméras, éclairage...)
  • Bibliothèques : Jquery, Nyromodal (pour les overlays dans le DOM), Modernizr (pour la détection des fonctionnalités du navigateur), Stats (pour l'affichage du nombre de FPS), Detector (pour afficher un message d'erreur si on force le lancement de la version webGL, sans supporter webGL (par défaut, la version flash se lance))

Vous pourrez trouver plus d'informations dans la salle "le bureau". C'est dans cette salle que vous pourrez visionner de la documentation, des présentations sous forme de diaporama HTML5, rapports...
